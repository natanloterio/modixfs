# AgentFS — Publishing

**Evidence source:** https://github.com/tursodatabase/agentfs README (SDK on npm/PyPI/crates.io)
**Rating:** — N/A

AgentFS does not define a publishing model for tools or capabilities. The project publishes its own SDK libraries (agentfs-sdk on npm, PyPI, and crates.io), which is straightforward package distribution for the storage library itself. However, there is no registry or packaging convention for sharing AgentFS-backed tool definitions with other teams or LLM frameworks. Publishing as a criterion for tool-integration discoverability does not apply to AgentFS's design scope.
